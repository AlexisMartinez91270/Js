Les images

Les images répondent à des standartds. On utilise les types suivants :

jpeg
Meilleur qualité pour le web mais poids importnats => temps de chargement
jpg
Idem jpeg mais plus léger. Meilleur option qualiuté (16 bits + densité forte)
gif
Plus vieux format standardisé. (256 couleurs seulement). Mauvaise qualité pour une image. Idéal pour un picto ou une icône. Préserve la transparence
png
Qualité du jpg mais préserve la transparence et plus léger que le jpg
svg
Saclable Vector Graphique : image vectorielle
